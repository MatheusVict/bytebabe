Atualmente, o ByteBabe CLI é desenvolvido exclusivamente para ambientes Unix-like (Linux/macOS).

Alternativas Disponíveis

1. Windows Subsystem for Linux (WSL)

  • Instale o WSL2

  • Use uma distribuição Linux compatível (Ubuntu recomendado)

  • Instale o ByteBabe normalmente dentro do WSL

2. Git Bash

  • Instale o Git for Windows

  • Use o Git Bash como terminal

  • Algumas funcionalidades podem não funcionar corretamente

Limitações Conhecidas

  • Sem suporte nativo para PowerShell

  • Sem instalador Windows (.exe/.msi)

  • Alguns comandos podem falhar no Git Bash

  • Dependências específicas do Linux necessárias

Próximos Passos

Por enquanto, recomendamos fortemente o uso do WSL2 para a melhor experiência possível no Windows.
