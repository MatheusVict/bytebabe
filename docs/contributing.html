Contributing 🤝

Como Contribuir

1. Setup Inicial

# Clone o repositório git clone https://github.com/mrpunkdasilva/bytebabe.git # Entre no diretório cd bytebabe # Dê permissão de execução chmod +x bin/bytebabe

2. Workflow Básico

  1. Fork o repositório

  2. Crie uma branch: feature/sua-feature

  3. Faça suas alterações

  4. Teste manualmente os comandos

  5. Envie um PR

3. Guidelines

  • Siga os padrões de código Shell

  • Teste todos os comandos

  • Documente mudanças

  • Use commits semânticos

Estrutura do Projeto

bytebabe/ ├── bin/ # Executável principal ├── commands/ # Comandos da CLI ├── lib/ # Funções compartilhadas └── scripts/ # Scripts utilitários
